# Speech Recognition Update [Desktop]
**Objective:** Allow the user to talk verbally to the AI
---
## New Features (v2.5.14.5)
### Voice Interaction (Desktop Mode Only)
- Added continuous speech recognition
- Implemented wake-word detection. saying the name guarantees an immediate response
- Introduced intelligent probabilistic responses:
  - Base chance for the AI to reply to anything you say (subtle background listening)
  - Higher chance to jump in
- Autonomous speaking behavior:
  - The AI can initiate conversation after periods of silence
  - Random comments appear occasionally when quiet
- All spoken responses use TTS output
- Smooth integration with existing mode switching — voice commands still work to switch back to Discord
