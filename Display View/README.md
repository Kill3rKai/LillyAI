# Display View Update
**Objective:** Give Lilly the ability to "see" my screen in desktop mode, enabling her to describe what’s currently visible
---
## New Features (v2.5.15)
### Screen Vision (Desktop Mode Only)
- Added real-time screen capture triggered by natural voice.
- Lilly can now:
  - Perform OCR (optical character recognition) to extract and read any visible text
  - Generate a basic description of what appears on screen, including detected text content
- Screen perception is fed directly into Lilly’s response system — she can comment on, react to, or assist with whatever is currently displayed (code editors, games, browsers, errors, memes, etc.)
- Fully voice-integrated: just speak the trigger phrase and she’ll look, describe, and reply out loud
