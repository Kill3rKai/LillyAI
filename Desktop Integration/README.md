# Desktop Integration Update
**Objective:** Enable seamless switching between Discord bot mode and a fully independent desktop application mode, Lilly to run directly on the my computer.
---
## New Features (v2.5.13)
### Desktop Mode Integration
- Added persistent mode selection to remember whether Lilly should run in Discord or Desktop mode across restarts
- Introduced a unified launcher that automatically starts the correct interface based on the saved mode
- Implemented seamless mode switching:
  - From Discord: Natural language commands (e.g., "switch to desktop") shut down Lilly and launch the desktop application
  - From Desktop: Commands (e.g., "switch to discord") update the mode and prepare for restart in Discord
- New standalone desktop interface:
  - Console-based direct interaction with immediate user input
  - Support for clean exit and mode-switching commands
---
## Version 2.5 Update — 2.5.13
### Added: Desktop Integration
This update brings full desktop application support, Lilly operate completely independently on my local machine. Switch instantly between the Discord bot experience and a dedicated desktop mode featuring local processing, direct console chat, and expressive voice output.
